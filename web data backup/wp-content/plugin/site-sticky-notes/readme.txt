Add sticky notes directly to website content.

== Description ==

Communicate and collaborate by leaving sticky notes on any page of your site. Drag and drop them anywhere you'd like, type a note, and change its color/priority.

This plugin is a fork of StickyNotate https://wordpress.org/plugins/stickynotate/

Here's how it works:

* While signed in, go to any page on your site
* Click the **Add Note** button in the admin bar at the top of the screen
* Click and grab the top of the note to move it anywhere on the page
* Type your note text
* Click the save button in the lower right of your note to keep your changes

== Installation ==

1. Upload `site-sticky-notes` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find Site Sticky Notes in your WordPress admin under **Site Notes**
